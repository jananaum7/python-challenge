#Financial Data Analysis

The Financial Data Analysis project involved examining the budget_data.csv file to extract crucial financial insights. The analysis covered 86 months and revealed a net profit/loss of $22,564,198. It also showed an average change in profits of $-8,311.11, with the highest profit increase in August 2016 ($1,862,002) and the steepest decline in February 2014 ($-1,825,558). I wrote a Python script to conduct this analysis, display the results, and export the findings to a text file, emphasizing the importance of data analysis in making informed financial decisions.

I received valuable assistance from the XPERT assistant for my code and from my tutor, who helped me learn how to export the results as a text file in Visual Studio Code.

Election Data Analysis
The Election Data Analysis project focused on processing the election_data.csv dataset to enhance the vote-counting system in a rural town. The main goals were to tally the total votes, evaluate the performance of each candidate, and determine the election winner. The analysis showed that 369,711 votes were cast, with Diana DeGette winning 73.812% of the votes. I utilized a Python script to analyze the data, calculate the necessary metrics, and present the results, demonstrating how data analysis can optimize electoral processes.
